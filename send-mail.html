ERGSN 브랜드 메일 발송

특정 바이어 / 파트너에게 ERGSN 브랜드 디자인이 적용된 메일을 직접 발송합니다. 발송된 메일은 사이트의 RFQ 자동 발송 메일과 동일한 헤더 / 푸터 / 디자인으로 도착합니다.

관리자 인증이 필요합니다. ADMIN_KEY (Cloudflare Worker secret 과 동일) 를 입력하세요. 인증은 브라우저 세션 동안만 유지됩니다.

발송 메일은 ERGSN 브랜드 헤더 + 푸터로 자동 wrapping 됩니다. From 은 반드시 ergsn.net 도메인이어야 합니다 (Resend SPF/DKIM 검증). Reply-To 는 답장 시 받을 곳입니다 (default: ceodon@gmail.com).
